# Computer Vision & AI Study Lab

Este repositório documenta minha rotina de estudos intensivos e experimentos práticos durante o Summer Break 2026. 

O objetivo é consolidar a base matemática e técnica necessária para atuar em pesquisa de ponta nas áreas de **Visão Computacional**, **Processamento Digital de Imagens** e **Deep Learning**.

## Objetivo
Construir um pipeline de conhecimento sólido, partindo da manipulação de dados matriciais até a implementação de redes neurais convolucionais (CNNs) e detectores de objetos (YOLO).

## Tópicos & Estrutura

O repositório está organizado conforme a evolução do cronograma de estudos:

### [01] Foundations (Data Science & Math)
- Manipulação avançada de matrizes e tensores (`numpy`).
- Visualização de dados e histogramas (`matplotlib`).
- Estatística aplicada a imagens.
- *Projetos:* Chessboard Matrix Logic, Image Channel Splitting.

### [02] Computer Vision (Classic)
- Pré-processamento de imagens (`opencv`).
- Transformações geométricas, Binarização e Thresholding.
- Filtros (Gaussian, Median) e Detecção de Bordas (Canny).
- *Projetos:* Real-time Webcam Filters, Document Scanner Logic.

### [03] Deep Learning & Neural Networks
- Fundamentos de Redes Neurais e Backpropagation.
- Arquitetura de CNNs (Convolução, Pooling, Stride).
- Implementação de modelos em `pytorch`.

## Tech Stack
* **Linguagem:** Python 3.x
* **Core:** NumPy, Pandas
* **Visão:** OpenCV (cv2)
* **DL Framework:** PyTorch
